## Let your agent use NEAR

AI Agents can easily control accounts and assets on all blockchains thanks to NEAR's unique features. You simply need to give them access to a NEAR account and they can start interacting with smart contracts, sending transactions, and managing assets everywhere.

## Build trustless multi-chain agents

The [Shade Agent Framework](/ai/shade-agents/introduction) enables you to build **verifiable** **multi-chain AI agents**. These agents operate in Trusted Execution Environments (TEEs) and leverage NEAR features such as multi-key management and Chain Signatures to securely manage assets and sign transactions across multiple blockchains.

Shade agents can autonomously sign transactions on any chain, interact with AI models and external data sources, and perform privacy-preserving, verifiable computations.

<Info>
  Shade Agents power **Agentic Protocols**: a new type of decentralized application designed to be autonomous, proactive, and intelligent.
</Info>

## Common questions

<AccordionGroup>
  <Accordion title="Can trustless agents lose access to their wallets?">
    Not with Shade Agents. They use decentralized key management—any instance of the agent with the same code can access the same accounts.
  </Accordion>
  
  <Accordion title="Do I need to know blockchain development?">
    Our tools and resources help to abstract away the complexity of blockchain development.
  </Accordion>
  
  <Accordion title="What chains can my agent interact with?">
    NEAR accounts allow you to transact on all chains, including Bitcoin, Ethereum and Solana thanks to Chain Signatures.
  </Accordion>
</AccordionGroup>
