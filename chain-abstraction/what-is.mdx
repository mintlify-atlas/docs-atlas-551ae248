Through a combination of innovative technologies, NEAR enables developers to build applications that work seamlessly across multiple blockchains while abstracting away the underlying complexity for both developers and end users.

## Core Technologies

NEAR's chain abstraction framework consists of three core technologies that work together to create seamless cross-chain experiences:

<CardGroup cols={3}>
  <Card title="Chain Signatures" icon="signature" href="/chain-abstraction/chain-signatures">
    Allow NEAR accounts and smart contracts to sign transactions for all other chains including Bitcoin, Ethereum, and Solana.
  </Card>
  <Card title="NEAR Intents" icon="arrows-rotate" href="/chain-abstraction/intents">
    Express desired outcomes and let a network of solvers compete to fulfill intents optimally across chains.
  </Card>
  <Card title="Omnibridge" icon="bridge" href="/chain-abstraction/omnibridge">
    Multi-chain bridge enabling secure and efficient cross-chain transfers with unified token management.
  </Card>
</CardGroup>

## Why Chain Abstraction Matters

By building on NEAR, developers do not need to worry about the complexities of integrating with multiple blockchains. Instead, they can focus on building great applications that work seamlessly across all chains.

Meanwhile, users can enjoy a smooth experience, using unified accounts and assets without needing to even understand on which blockchain they are operating.

<AccordionGroup>
  <Accordion title="Benefits for Developers">
    - Integrate with multiple blockchains through a single NEAR API
    - Focus on application logic instead of blockchain complexity
    - Reach users regardless of their preferred blockchain network
    - Build cross-chain dApps without managing multiple integrations
  </Accordion>
  <Accordion title="Benefits for Users">
    - Operate across all chains using a single NEAR account
    - Access assets and services from multiple blockchains seamlessly
    - Enjoy a unified and intuitive user experience
    - No need to manage multiple wallets or understand blockchain specifics
  </Accordion>
</AccordionGroup>

## Example Use Case

<Accordion title="Cross-Chain NFT Marketplace">
  Imagine building a digital art marketplace where users can purchase NFTs from different blockchains (Ethereum, Solana, etc.). Without chain abstraction, you'd need to:

  - Implement multiple blockchain connections
  - Handle different wallet types
  - Manage cross-chain transfers
  - Build complex UIs to explain blockchain concepts

  With chain abstraction, both you and your users just focus on the core experience: browsing and trading art. All blockchain complexity is handled automatically behind the scenes.
</Accordion>

## How It Works Together

These technologies work in concert to provide a complete chain abstraction solution:

<Steps>
  <Step title="Chain Signatures for Control">
    Your NEAR account can control addresses and sign transactions on any blockchain, giving you multi-chain account ownership from a single point.
  </Step>
  <Step title="Intents for Execution">
    Instead of manually executing complex cross-chain transactions, you express what you want (an intent), and solvers compete to fulfill it optimally.
  </Step>
  <Step title="Omnibridge for Asset Movement">
    When you need to move assets between chains, Omnibridge handles the secure transfer using Chain Signatures and chain-specific verification.
  </Step>
</Steps>

<Note>
  Each technology can be used independently, but they're designed to work together for the most powerful chain abstraction capabilities.
</Note>
