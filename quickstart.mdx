## Overview

This guide will help you get started with NEAR Protocol quickly. You'll learn how to create an account, get testnet tokens, and interact with the blockchain.

<Note>
  This quickstart focuses on testnet development. Once you're comfortable, you can deploy to mainnet.
</Note>

## Getting Started

<Steps>
  <Step title="Create a NEAR Account">
    You'll need a NEAR account to interact with the blockchain. NEAR uses human-readable account names like `alice.near` instead of cryptographic hashes.

    ### Testnet Account (Recommended for Development)

    Create a testnet account for free:

    1. Visit the [NEAR Testnet Wallet](https://testnet.mynearwallet.com/)
    2. Click "Create Account"
    3. Choose your account name (e.g., `yourname.testnet`)
    4. Secure your account with a passphrase or recovery method

    <Tip>
      Testnet accounts end with `.testnet` and are perfect for development and testing without spending real tokens.
    </Tip>

    ### Mainnet Account (For Production)

    For production applications:

    1. Visit [MyNEARWallet](https://app.mynearwallet.com/)
    2. Follow the account creation flow
    3. You can create an account with social login (Google, email, etc.)
    4. Or use Telegram via [HERE Wallet bot](https://web.telegram.org/k/#@herewalletbot)

    <Warning>
      Keep your recovery phrase or seed phrase secure. This is the only way to recover your account if you lose access.
    </Warning>
  </Step>

  <Step title="Get Testnet Tokens">
    You'll need testnet NEAR tokens to pay for transactions and deploy contracts.

    ### Using the Faucet

    Visit the [faucet page](/faucet) or use one of these options:

    1. **Built-in Faucet**: Input your testnet account name and request tokens
    2. **External Faucet**: Visit [near-faucet.io](https://near-faucet.io/) for NEAR and other tokens (USDT, USDC, etc.)

    <Info>
      The faucet provides free testnet tokens. You'll typically receive 10-20 testnet NEAR tokens per request.
    </Info>
  </Step>

  <Step title="Install NEAR CLI">
    The NEAR CLI is a command-line tool for interacting with the NEAR blockchain.

    ```bash
    npm install -g near-cli
    ```

    Verify the installation:

    ```bash
    near --version
    ```

    <Tip>
      You can also use `npx near-cli` to run commands without installing globally.
    </Tip>
  </Step>

  <Step title="Login to Your Account">
    Connect NEAR CLI to your account:

    ```bash
    near login
    ```

    This will:
    1. Open a browser window
    2. Ask you to authorize NEAR CLI
    3. Save your credentials locally

    For testnet development, use:

    ```bash
    export NEAR_ENV=testnet
    near login
    ```

    <Note>
      The `NEAR_ENV` environment variable determines which network (testnet or mainnet) you're connected to.
    </Note>
  </Step>

  <Step title="Check Your Balance">
    Verify your account has tokens:

    ```bash
    near state yourname.testnet
    ```

    This will display:
    - Your account balance
    - Storage usage
    - Account status

    Example output:
    ```json
    {
      "amount": "100000000000000000000000000",
      "block_height": 123456789,
      "storage_usage": 182,
      "formattedAmount": "100"
    }
    ```
  </Step>

  <Step title="Make Your First Transaction">
    Try sending testnet tokens to another account:

    ```bash
    near send yourname.testnet recipient.testnet 1
    ```

    This sends 1 NEAR token from your account to `recipient.testnet`.

    <Tip>
      Transactions on NEAR finalize in approximately 1.3 seconds and cost less than 1¢ in fees.
    </Tip>
  </Step>
</Steps>

## Common Commands Reference

Here are some useful NEAR CLI commands:

<CodeGroup>

```bash View Account
# Check account state
near state yourname.testnet

# View account details
near view-state yourname.testnet --finality final
```

```bash Send Tokens
# Send NEAR tokens
near send sender.testnet receiver.testnet 5

# Send with a specific amount in yoctoNEAR
near send sender.testnet receiver.testnet 5000000000000000000000000 --yocto
```

```bash Contract Interactions
# Call a contract method (change state)
near call contract.testnet method_name '{"arg": "value"}' --accountId yourname.testnet

# View a contract method (read-only)
near view contract.testnet view_method '{"arg": "value"}'
```

```bash Account Management
# Create a sub-account
near create-account sub.yourname.testnet --masterAccount yourname.testnet --initialBalance 10

# Delete an account
near delete-account yourname.testnet beneficiary.testnet
```

</CodeGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="NEAR CLI not found after installation">
    Make sure your npm global bin directory is in your PATH:
    
    ```bash
    npm config get prefix
    ```
    
    Add the bin directory to your PATH in `.bashrc`, `.zshrc`, or equivalent:
    
    ```bash
    export PATH="$PATH:$(npm config get prefix)/bin"
    ```
  </Accordion>

  <Accordion title="Faucet not working">
    If the faucet is not responding:
    
    1. Try the alternative faucet at [near-faucet.io](https://near-faucet.io/)
    2. Report the issue on [NEAR Telegram Channel](https://t.me/neardev)
    3. Ask in the community for testnet tokens
  </Accordion>

  <Accordion title="Transaction failed">
    Common reasons for transaction failures:
    
    - **Insufficient balance**: Make sure you have enough tokens for the transaction and gas fees
    - **Invalid account**: Verify the recipient account exists
    - **Network issues**: Check your connection and try again
    - **Gas limit exceeded**: Complex transactions may need more gas
  </Accordion>

  <Accordion title="Wrong network (testnet vs mainnet)">
    Always verify which network you're using:
    
    ```bash
    # For testnet
    export NEAR_ENV=testnet
    
    # For mainnet
    export NEAR_ENV=mainnet
    ```
    
    Add this to your shell profile to make it persistent.
  </Accordion>
</AccordionGroup>
